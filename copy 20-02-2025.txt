# ğŸ“œ uninstall_nginx.yaml

Create file:

```bash
nano uninstall_nginx.yaml
```

Paste this **exact** YAML:

```yaml
- name: Uninstall Nginx and Clean Up Configuration
  hosts: webservers
  become: true

  tasks:
    - name: Stop Nginx service if running
      service:
        name: nginx
        state: stopped
      ignore_errors: true

    - name: Disable Nginx service
      service:
        name: nginx
        enabled: no
      ignore_errors: true

    - name: Uninstall Nginx packages
      yum:
        name:
          - nginx
          - nginx-core
          - nginx-filesystem
          - nginx-mimetypes
        state: absent
      ignore_errors: true

    - name: Remove Nginx configuration folder
      file:
        path: /etc/nginx
        state: absent
      ignore_errors: true

    - name: Remove default web content
      file:
        path: /usr/share/nginx/html
        state: absent
      ignore_errors: true
```
